Excel-export controleren

Een script leest een export in, markeert regels met ontbrekende marge of prijs en schrijft alleen de uitzonderingen naar een controlebestand.

  • Handig bij productlijsten, orderexports en voorraadbestanden.
  • De gebruiker controleert alleen de regels die aandacht nodig hebben.
import pandas as pd

producten = pd.read_excel("producten.xlsx")
fouten = producten[
    producten["prijs"].isna() |
    (producten["marge"] < 0.25)
]

fouten.to_excel("controle_producten.xlsx", index=False)

API-data ophalen

Python kan gegevens uit een API ophalen, normaliseren en klaarzetten voor database, dashboard of Excel.

  • Geschikt voor orders, voorraad, tickets en productdata.
  • Met logging zie je direct of een koppeling is gelukt.
import requests

response = requests.get(
    "https://api.voorbeeld.nl/orders",
    headers={"Authorization": "Bearer TOKEN"},
    timeout=30,
)
response.raise_for_status()
orders = response.json()["orders"]
